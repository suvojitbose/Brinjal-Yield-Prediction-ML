# Brinjal-Yield-Prediction-ML
Machine learning scripts and multi-environment datasets for dual-target yield and biochemical stability prediction in Solanum melongena.
# Predictive Modelling for Yield and Biochemical Stability in Brinjal

This repository contains the dataset and Python source code required to reproduce the machine learning and multi-environment statistical framework (AMMI/GGE) presented in the manuscript: *"Predictive Modelling for Yield and Biochemical Stability in Brinjal (Solanum melongena L.) under Climatic Stress."*

### Files Included:
* `AMMI FINAL DATA (1).xlsx`: The complete raw dataset including morphological, biochemical, and weather parameters.
* `Brinjal_Yield_Prediction_Models.ipynb`: The Google Colab notebook containing the complete predictive pipeline, including Random Forest, Stepwise Multiple Linear Regression (SWMLR), Decision Trees, and AMMI/GGE stability visualizations.
